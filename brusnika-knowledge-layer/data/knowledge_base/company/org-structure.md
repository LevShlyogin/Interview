# Организационная структура «БРУСНИКА»

Описание управленческой структуры компании: блоки, ключевые подразделения и линии подчинения. Документ для сотрудников и руководителей при навигации по процессам и эскалации вопросов.

## Верхний уровень

**Генеральный директор** — единый центр управления. Под ним восемь функциональных блоков и региональная матрица в СГП и продажах.

## Блок девелопмента

| Подразделение | Функция |
|---------------|---------|
| Управление проектами | Stage-gate, календарь проекта, координация участков |
| Отдел земельных участков | Поиск, оценка, сделки по земле |
| Отдел проектирования | АР, КР, ИРД, согласования с экспертизой |

**Руководитель блока:** Директор по девелопменту. Инициирует ТЭО и feasibility, утверждает переход между воротами stage-gate.

## Служба генерального подрядчика (СГП)

| Подразделение | Функция |
|---------------|---------|
| Управление строительства по регионам | 5 региональных директоров СГП |
| Отдел ПТО | Исполнительная документация, объёмы, сметы |
| Отдел снабжения и закупок | Материалы, тендеры, логистика |
| Охрана труда и ТБ | Допуски, инструктажи, расследования |
| Служба контроля качества (СКК) | Входной контроль, приёмка работ, дефекты |

**Руководитель:** Директор СГП. Отчитывается генеральному директору.

## Коммерческий блок

- Отдел продаж (региональные офисы продаж)
- Отдел маркетинга
- Отдел ипотеки и партнёрских программ

## Финансовый блок

- Казначейство и проектное финансирование
- Бухгалтерия
- Отдел бюджетирования

## Прочие блоки

- **Юридический блок** — ДДУ, земля, претензии
- **HR-блок** — подбор, онбординг, грейды, обучение
- **IT-блок** — системы, доступы, ИБ
- **Блок эксплуатации** — управляющая компания (УК)

## Матрица по регионам

В каждом из 5 регионов есть: директор проекта (девелопмент), директор СГП, руководитель продаж. Они координируются на еженедельном региональном комитете.
