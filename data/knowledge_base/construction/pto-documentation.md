# Документация производственно-технического отдела (ПТО)

Документ регламентирует состав, порядок оформления, ведения и хранения исполнительной документации производственно-технического отдела (ПТО) на стройплощадках «БРУСНИКА». Обязателен для инженеров ПТО, прорабов и начальников участков.

## Назначение ПТО на участке

ПТО обеспечивает:

- Техническое сопровождение СМР в соответствии с РД и ППР
- Ведение исполнительной документации (ИД)
- Формирование актов скрытых работ (АОСР) и исполнительных схем
- Подготовку комплекта ИД для получения ЗОС
- Взаимодействие с органами строительного надзора

На каждом участке — минимум **2 инженера ПТО** (старший + инженер).

## Состав исполнительной документации

### Общестроительные работы

- АОСР (акты освидетельствования скрытых работ)
- Исполнительные геодезические схемы
- Паспорта на материалы (копии из входного контроля)
- Протоколы лабораторных испытаний
- Журналы бетонных работ, сварочных работ

### Инженерные системы

- Исполнительные схемы разводки
- Протоколы опрессовки, промывки, прогона
- Паспорта оборудования
- Акты индивидуальных испытаний

### Пусконаладка

- Протоколы наладки систем
- Акты готовности к комплексному опробованию
- Журнал пусконаладочных работ

## Порядок оформления АОСР

1. Прораб подаёт заявку на освидетельствование за **24 часа**
2. ПТО проверяет комплектность (фото, паспорта, соответствие РД)
3. Комиссия (ПТО + прораб + при необходимости ГИП) проводит осмотр
4. АОСР подписывается в день осмотра
5. Копия АОСР прикрепляется к разделу ИД в «ДокПТО»

**Правило:** задним числом АОСР не оформляются. Нарушение — дисциплинарное взыскание + запись в KPI.

## Комплект ИД для ЗОС

Комплект формируется за **60 рабочих дней** до плановой даты ЗОС. Проверка:

- Полнота разделов (чек-лист ZOS-CHECK-2025)
- Соответствие РД и факта
- Подписи всех ответственных лиц

Передача комплекта в надзорный орган — через менеджера по разрешениям.

## Практический элемент: график сдачи ИД

| Месяц строительства | Действие ПТО |
|---------------------|--------------|
| Ежемесячно | Сверка ИД с фактом выполненных работ |
| −4 мес. до ЗОС | Аудит комплектности, план закрытия пробелов |
| −2 мес. до ЗОС | Предъявление ИД внутренней комиссии |
| −1 мес. до ЗОС | Передача в надзор, устранение замечаний |
| Дата ЗОС | Финальный комплект подписан |
