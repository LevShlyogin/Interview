# История и ключевые вехи «БРУСНИКА»

Краткая история компании и основные вехи развития. Полезен для онбординга и внешних коммуникаций.

## Хронология

| Год | Событие |
|-----|---------|
| 2004 | Основание компании, первый жилой проект в Тюмени (1 дом, 12 000 м²) |
| 2008 | Выход в Екатеринбург, запуск отдела проектирования |
| 2012 | Создание собственной СГП — отказ от внешних генподрядчиков |
| 2015 | Вход в Московский регион, проектное финансирование через эскроу |
| 2017 | Санкт-Петербург, бизнес-класс в портфеле |
| 2019 | Новосибирск, 10-й региональный офис продаж |
| 2021 | Портфель 15+ проектов, 500 000 м² в строительстве |
| 2023 | Цифровизация: единая ERP для СГП и девелопмента |
| 2025 | 800 000 м² в строительстве, ~6 500 сотрудников |

## Ключевые решения

**Собственная СГП (2012)** — переломный момент. До этого компания работала с внешними генподрядчиками и столкнулась с разрывом сроков на 2 проектах. Решение строить своими силами принял совет директоров единогласно.

**Эскроу (2015)** — переход на проектное финансирование по 214-ФЗ с полным раскрытием для дольщиков.

**Stage-gate (2018)** — формализация ворот проекта; без прохождения Gate 3 (РНС) строительство не стартует.
