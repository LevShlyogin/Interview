# Процесс пусконаладки

Документ регламентирует порядок выполнения пусконаладочных работ (ПНР), комплексного опробования инженерных систем и подготовки объекта к получению заключения о соответствии (ЗОС) и последующей передаче. Обязателен для инженеров ПТО, начальников участков, специалистов по наладке и управляющей компании.

## Место в жизненном цикле

Пусконаладка — **обязательный этап после завершения СМР** и **перед** передачей квартир клиентам и передачей объекта в эксплуатацию УК:

```
СМР завершены → Пусконаладка → ЗОС → Передача клиентам → Передача УК
```

Без завершения ПНР и получения ЗОС передача квартир по ДДУ и приёмка УК не начинаются.

## Этапы пусконаладки

### Этап 1. Индивидуальные испытания (за 60 дней до ЗОС)

По каждой инженерной системе отдельно:

- Электроснабжение: измерение сопротивления изоляции, проверка УЗО
- Водоснабжение и канализация: опрессовка, прогон воды
- Отопление: гидравлические испытания, промывка
- Вентиляция: замер воздухообмена
- Лифты: индивидуальные испытания (орган надзора)
- Слаботочные системы: ДУ, СКУД, домофон

Ответственный: инженер наладки + представитель субподрядчика. Результат — протоколы испытаний в ИД.

### Этап 2. Комплексное опробование (за 30 дней до ЗОС)

Совместная работа всех систем в режиме, близком к эксплуатационному:

- Проверка автоматики (ИТП, насосы, вентиляция)
- Имитация аварийных режимов
- Проверка резервного питания (дизель-генератор)
- Тепловизионный контроль ограждающих конструкций

Комиссия: начальник участка, ПТО, ОТК, представитель УК, ГИП (при необходимости).

### Этап 3. Подготовка к ЗОС (за 14 дней)

- Комплект ИД сдан (см. [Документация ПТО](pto-documentation.md))
- Устранены замечания внутренней комиссии
- Благоустройство: подъезды, освещение, дороги — готовность ≥ 80%
- Подача документов в надзорный орган

### Этап 4. Получение ЗОС

Заключение о соответствии (ЗОС) — разрешение на ввод объекта в эксплуатацию. Срок выдачи надзором: **до 7 рабочих дней** после приёмки.

## Взаимодействие с УК и продажами

За **45 дней** до плановой даты ЗОС начальник участка уведомляет:

- **УК** — для подготовки приёмки (см. [Приёмка от строительства](../exploitation/handover-from-construction.md))
- **Коммерческий блок** — для планирования передачи квартир (см. [Передача клиенту](../commercial/handover-to-client.md))

## Практический элемент: график ПНР типового жилого дома

| Неделя до ЗОС | Работы | Ответственный |
|---------------|--------|---------------|
| −8 | Индивидуальные испытания: электрика, водоснабжение | Инженер наладки |
| −7 | Индивидуальные испытания: отопление, вентиляция | Инженер наладки |
| −6 | Лифты: испытания с органом надзора | Субподрядчик + ПТО |
| −5 | Слаботочные системы | Субподрядчик |
| −4 | Комплексное опробование, суточный прогон | Комиссия |
| −3 | Устранение замечаний | Прорабы, субподряд |
| −2 | Аудит ИД, подача на ЗОС | ПТО |
| −1 | Получение ЗОС (план) | Менеджер по разрешениям |
| 0 | Старт передачи квартир / приёмка УК | Коммерция + УК |
