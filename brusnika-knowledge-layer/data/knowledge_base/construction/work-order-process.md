# Наряд-заказ на производство работ

Регламент описывает порядок формирования, согласования и закрытия наряд-заказов (НЗ) на выполнение строительно-монтажных работ на объектах «БРУСНИКА». Документ обязателен для начальников участков, прорабов, мастеров и субподрядчиков.

## Назначение наряд-заказа

Наряд-заказ — первичный документ оперативного управления производством работ. Фиксирует: что, где, когда, кем и в каком объёме должно быть выполнено. Без действующего НЗ работы к оплате не принимаются.

## Типы наряд-заказов

| Тип | Срок | Инициатор |
|-----|------|-----------|
| НЗ-неделя | 5 рабочих дней | Прораб |
| НЗ-смена | 1 смена (12 ч) | Мастер |
| НЗ-субподряд | По договору | Прораб + субподрядчик |

## Порядок оформления

### Шаг 1. Формирование (прораб)

Ежедневно до **16:00** прораб формирует НЗ на следующий день в мобильном приложении «СтройНЗ». Указывает: вид работ, участок, объём, бригаду/субподряда, нормативные часы.

### Шаг 2. Согласование (начальник участка)

Начальник участка утверждает НЗ до **17:30**. Проверяет соответствие КСГ, наличие РД, допусков и материалов.

### Шаг 3. Исполнение

Бригадир/представитель субподрядчика отмечает факт выполнения в приложении. Фотофиксация ключевых этапов — обязательна.

### Шаг 4. Приёмка (мастер / прораб)

Мастер принимает работы в день выполнения. При замечаниях — статус «Возврат», повторная сдача на следующий день.

### Шаг 5. Закрытие (ПТО)

Инженер ПТО закрывает НЗ в системе, данные уходят в учёт объёмов для КС-2.

## Требования к субподрядным НЗ

Субподрядчик обязан назначить ответственного представителя с правом подписи в «СтройНЗ». НЗ подписывается электронной подписью обеих сторон. Споры по объёмам — в течение **2 рабочих дней** через начальника участка.

## Практический элемент: чек-лист прораба перед выпуском НЗ

- [ ] Работы есть в КСГ текущей недели
- [ ] РД на участке работ актуальна (штамп ПТО)
- [ ] Материалы на складе (остаток проверен)
- [ ] Бригада допущена (удостоверения, инструктаж ОТ)
- [ ] ППР содержит данный вид работ
- [ ] Предыдущий НЗ на этом участке закрыт
