# Шаблон технико-экономического обоснования (ТЭО)

Документ определяет структуру, требования к данным и порядок подготовки технико-экономического обоснования (ТЭО) для девелоперских проектов «БРУСНИКА». **Уровень доступа: restricted** — только для финансового блока, руководителей девелопмента и членов Инвестиционного комитета.

## Назначение ТЭО

ТЭО — ключевой артефакт для принятия инвестиционного решения на воротах G2. Содержит финансовую модель проекта, анализ чувствительности, оценку рисков и рекомендацию по входу в сделку.

Срок подготовки: **45–60 рабочих дней** с момента завершения due diligence.

## Структура документа

### Раздел 1. Резюме для руководства (Executive Summary)

- Описание проекта, локация, класс жилья
- Ключевые показатели: IRR, NPV, срок окупаемости, маржинальность
- Рекомендация: инвестировать / не инвестировать / инвестировать с условиями

### Раздел 2. Описание проекта

- Земельный участок: площадь, ВРИ, ГПЗУ
- ТЭП: площадь застройки, количество квартир, машино-места
- Календарный план: проектирование, строительство, продажи

### Раздел 3. Рыночный анализ

- Конкурентное окружение (радиус 3 км)
- Ценовой коридор и темпы продаж
- Доля ипотечных сделок в регионе

### Раздел 4. Финансовая модель

- Доходы: прайс-лист (ссылка на [Ценовая политика](../commercial/pricing-policy.md), confidential)
- Расходы: земля, проектирование, СМР, финансирование, коммерция
- Налоги, эскроу, проектное финансирование
- Сценарии: базовый, оптимистичный, пессимистичный

### Раздел 5. Анализ чувствительности

Варьируются: цена продажи (±5%), себестоимость СМР (±7%), темпы продаж (±20%), ключевая ставка (±2 п.п.).

### Раздел 6. Риски и митигация

Матрица рисков: вероятность × влияние. Обязательные категории: разрешительные, строительные, рыночные, финансовые, юридические.

## Пороговые показатели

| Показатель | Комфорт-класс | Бизнес-класс |
|------------|---------------|--------------|
| IRR (без учёта земли) | ≥ 20% | ≥ 18% |
| IRR (полный) | ≥ 18% | ≥ 15% |
| NPV (10% ставка) | > 0 | > 0 |
| Срок окупаемости | ≤ 7 лет | ≤ 8 лет |
| Маржа EBITDA | ≥ 22% | ≥ 18% |

## Практический элемент: чек-лист перед подачей на ИК

- [ ] Финмодель в утверждённом шаблоне Excel (версия FM-2025.1)
- [ ] Данные прайс-листа согласованы с коммерческим блоком
- [ ] Смета СМР верифицирована директором СГП
- [ ] Ставки проектного финансирования подтверждены банком-партнёром
- [ ] Юридический DD приложен и подписан руководителем юр. блока
- [ ] Сценарный анализ содержит минимум 3 сценария
- [ ] Резюме на 2 страницы подготовлено для членов ИК

**Хранение:** SharePoint `/Restricted/TEO/{регион}/{проект}/`. Запрещено пересылать по email.
