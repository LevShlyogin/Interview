# Передача квартир клиенту

Документ регламентирует процесс осмотра, оформления и передачи квартир дольщикам после завершения строительства и получения ЗОС. Применяется менеджерами по передаче, отделом продаж, СГП и управляющей компанией.

## Предусловия передачи

Передача квартир клиентам начинается **только после**:

1. Завершения [пусконаладки](../construction/commissioning-process.md)
2. Получения **ЗОС** (заключение о соответствии)
3. Регистрации права собственности на объект в Росреестре (для всего дома)
4. Готовности квартиры по чек-листу сдачи (ОТК + менеджер по передаче)

## Этапы передачи

### 1. Уведомление клиента

За **30 дней** до плановой даты передачи клиент получает SMS/email с предложением записаться на осмотр. Повторное уведомление — за 7 дней.

### 2. Запись на осмотр

Клиент записывается через личный кабинет или call-центр. Слоты: **45 минут** на квартиру. Менеджер по передаче + представитель СГП (при необходимости).

### 3. Осмотр квартиры

Совместный осмотр по чек-листу (отделка, окна, сантехника, электрика, счётчики). Замечания фиксируются в акте осмотра и системе «ДефектСГП» (см. [Управление дефектами](../construction/defect-management.md)).

| Категория замечания | Влияние на подписание акта |
|---------------------|---------------------------|
| Критическое / значительное | Акт не подписывается до устранения |
| Незначительное | Подписание с перечнем замечаний |
| Косметическое | Подписание, устранение в гарантийный срок |

### 4. Подписание акта приёма-передачи

При отсутствии блокирующих замечаний клиент подписывает акт. Передаются: ключи, паспорт квартиры, инструкции по эксплуатации, показания счётчиков.

### 5. Регистрация перехода

Данные передаются в УК для открытия лицевого счёта и начала обслуживания. Гарантийный период — **5 лет** (см. [Гарантийные обращения](../exploitation/warranty-claims.md)).

## Сроки

| Показатель | Целевое значение |
|------------|------------------|
| Начало передачи после ЗОС | ≤ 14 рабочих дней |
| Устранение значительных дефектов | ≤ 10 рабочих дней |
| % квартир переданных за 90 дней после старта | ≥ 85% |

## Практический элемент: чек-лист готовности квартиры

- [ ] Отделка завершена по спецификации ДДУ
- [ ] Окна, двери — исправны
- [ ] Сантехника и электрика — проверены
- [ ] Счётчики установлены, опломбированы
- [ ] Уборка выполнена
- [ ] Ключи и домофон — готовы
- [ ] Паспорт квартиры сформирован
