# Соответствие 214-ФЗ

Документ описывает ключевые требования Федерального закона № 214-ФЗ «Об участии в долевом строительстве многоквартирных домов и иных объектов недвижимости» и меры обеспечения compliance в «БРУСНИКА». Для юридического блока, девелопмента, коммерции и финансов.

## Ключевые обязательства застройщика

### 1. Проектная декларация

- Размещение на наш.дом.рф **до начала продаж**
- Обновление при существенных изменениях (в течение **5 рабочих дней**)
- Ответственный: менеджер по разрешениям + юрист

### 2. Договор ДДУ

- Только типовая форма, утверждённая Минстроем (с дополнениями по 214-ФЗ)
- Обязательное указание срока передачи объекта
- Регистрация в Росреестре

### 3. Эскроу-счета

С 2019 года — единственная схема привлечения средств дольщиков. См. [Эскроу-счета](../finance/escrow-accounts.md).

### 4. Раскрытие информации

- Сайт застройщика — актуальная декларация, разрешения (РНС), ход строительства
- Уведомление дольщиков об изменении сроков — **за 30 дней** (при задержке > 3 мес. — согласие 2/3)

### 5. Передача объекта

- Передача квартиры — не позднее срока в ДДУ
- Акт приёма-передачи, устранение замечаний
- Компенсация неустойки при просрочке — 1/300 ставки ЦБ за день (с 2019)

## Организация compliance

| Функция | Ответственный | Периодичность |
|---------|---------------|---------------|
| Аудит проектных деклараций | Юр. блок | Ежеквартально |
| Проверка ДДУ на соответствие | Юрист по ДДУ | Каждый договор |
| Контроль эскроу | Финансы | Еженедельно |
| Мониторинг сроков передачи | Коммерция + СГП | Ежемесячно |
| Взаимодействие с Фондам защиты прав дольщиков | Юр. блок | По запросам |

## Риски и санкции

Нарушение 214-ФЗ влечёт:

- Штрафы для юрлица (до 1 млн ₽)
- Включение в реестр недобросовестных застройщиков
- Требование Фонда о возврате средств дольщикам
- Уголовная ответственность (при грубых нарушениях)

## Практический элемент: чек-лист compliance перед стартом продаж

- [ ] Проектная декларация зарегистрирована на наш.дом.рф
- [ ] РНС получен и опубликован
- [ ] Эскроу-счёт открыт в аккредитованном банке
- [ ] Шаблон ДДУ актуален (версия LEGAL-DDU-2025)
- [ ] Сайт содержит обязательную информацию
- [ ] Ответственные за обновление декларации назначены
