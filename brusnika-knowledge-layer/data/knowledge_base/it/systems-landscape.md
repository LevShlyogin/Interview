# IT-ландшафт компании

Документ даёт обзор корпоративных информационных систем «БРУСНИКА», их назначения, владельцев и правил доступа. Для всех сотрудников при онбординге и при работе с IT-сервисами.

## Принципы IT-ландшафта

- **Единый источник правды** по финансам и HR — ERP 1С
- **Специализированные системы** по доменам (стройка, продажи, проектирование)
- **Интеграция через шину** (ESB) для ключевых потоков данных
- **AD (Active Directory)** — единая аутентификация офиса

## Интеграции (ключевые потоки)

- CRM «Продажи» → 1С (сделки, оплаты)
- ЗакупкиСГП → 1С (закупки, платежи)
- Стройконтроль → 1С (объёмы, КС-2)
- HR-портал → AD (создание/блокировка УЗ)
