# Служба генерального подрядчика (СГП) — обзор

«БРУСНИКА» ведёт строительство исключительно силами собственной **Службы генерального подрядчика (СГП)** — без привлечения внешних генеральных подрядчиков. СГП обеспечивает контроль сроков, качества и себестоимости на всех стройплощадках компании.

## Масштаб и география

СГП работает во всех пяти регионах присутствия «БРУСНИКА»: Москва, Санкт-Петербург, Екатеринбург, Тюмень, Новосибирск. На стройплощадках одновременно занято около **4 000 человек**; управляющий аппарат СГП — **350 сотрудников**.

## Структура СГП

- **Региональные дирекции СГП** — управление стройками региона
- **Стройучастки** — начальник участка, прорабы, мастера, ПТО, ОТиТБ
- **Центр компетенций** — типовые решения, нормативы, обучение, KPI

## Ключевые функции

| Функция | Описание |
|---------|----------|
| Планирование СМР | Календарно-сетевые графики, ППР |
| Производство работ | Собственные бригады + субподряд |
| Закупки | Материалы и оборудование (см. [Закупки](procurement-process.md)) |
| Качество | Входной контроль, ОТК, дефект-листы |
| Безопасность | ОТиТБ на площадке |
| Ввод | Пусконаладка, получение ЗОС |
