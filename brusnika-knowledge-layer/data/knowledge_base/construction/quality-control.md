# Контроль качества строительства

Документ описывает трёхуровневую систему контроля качества строительно-монтажных работ на объектах «БРУСНИКА». Распространяется на собственные бригады СГП и субподрядчиков. Обязателен для инженеров ОТК, прорабов, начальников участков.

## Уровни контроля

### Уровень 1. Входной контроль

Контроль материалов и изделий при поступлении на склад. Подробности — в [Приёмке материалов](materials-acceptance.md).

### Уровень 2. Операционный контроль

Контроль в процессе выполнения работ. Мастер и прораб проверяют соответствие РД, ППР и нормативам **до** закрытия наряд-заказа.

**Скрытые работы** (армирование, гидроизоляция, разводка сетей) — оформление акта освидетельствования (АОСР) **до** закрытия последующих работ. Без АОСР работы считаются невыполненными.

### Уровень 3. Приёмочный контроль

Инженер ОТК проводит выборочную и полную приёмку завершённых видов работ. Формирует акты КС-2, участвует в комиссионных осмотрах.

## Ключевые контрольные точки

| Конструкция / система | Тип контроля | Ответственный |
|-----------------------|--------------|---------------|
| Свайное поле / фундамент | 100% + лаборатория | ОТК + лаборатория |
| Монолитный каркас | 100% геометрия, выборочно прочность | ОТК |
| Фасад | Выборочно 30%, тепловизор | ОТК + субподрядчик |
| Инженерные сети | 100% опрессовка/прогон | ОТК + наладчик |
| Отделка квартир | 100% по чек-листу сдачи | ОТК + прораб |

## Нормативная база

- СП и ГОСТ, указанные в РД проекта
- Внутренние стандарты СГП (ST-SGP-2025)
- Требования авторского надзора ГИПа

Отступление от проекта — только по протоколу изменений (см. [Управление проектированием](../development/design-management.md)).

## Практический элемент: чек-лист ОТК для монолитных работ

- [ ] Армирование соответствует РД (диаметр, шаг, вязка)
- [ ] АОСР на армирование подписан
- [ ] Опалубка геометрически верна (допуск ±5 мм)
- [ ] Закладные детали установлены
- [ ] Бетон марки не ниже проектной (паспорт партии)
- [ ] Уход за бетоном организован (температура, влажность)
- [ ] Акт промера прочности (7/28 суток) в графике
