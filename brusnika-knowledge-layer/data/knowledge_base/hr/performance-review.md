# Оценка эффективности

Документ регламентирует ежегодный цикл оценки эффективности (Performance Review) сотрудников «БРУСНИКА». Для руководителей, HR и сотрудников. Результаты оценки влияют на премию, повышение грейда и план развития.

## Цикл оценки

| Этап | Срок | Участники |
|------|------|-----------|
| Постановка целей | Январь–февраль | Сотрудник + руководитель |
| Промежуточный review | Июль | Сотрудник + руководитель |
| Самооценка | Октябрь | Сотрудник |
| Оценка руководителя | Ноябрь | Руководитель |
| Калибровка | Ноябрь–декабрь | Руководители + HR |
| Обратная связь | Декабрь | Сотрудник + руководитель |

## Структура оценки

Оценка состоит из двух блоков с равным весом:

### 1. Достижение целей (50%)

- 3–5 целей SMART на год
- Каждая цель с весом и шкалой 1–5
- Для СГП — связь с [KPI строительства](../construction/construction-kpi.md)

### 2. Компетенции (50%)

| Компетенция | Описание |
|-------------|----------|
| Результативность | Выполнение задач в срок и с качеством |
| Командность | Взаимодействие, обратная связь |
| Лидерство | Инициатива, развитие команды (для M-грейдов) |
| Клиентоориентированность | Внутренний/внешний клиент |
| Развитие | Обучение, новые навыки |

## Шкала итоговой оценки

| Балл | Рейтинг | Распределение (целевое) |
|------|---------|-------------------------|
| 4,5–5,0 | Выдающийся (A) | ≤ 10% |
| 3,8–4,4 | Превосходит (B) | ~ 25% |
| 3,0–3,7 | Соответствует (C) | ~ 55% |
| 2,0–2,9 | Требует улучшения (D) | ~ 8% |
| < 2,0 | Не соответствует (E) | ≤ 2% |

Калибровочная сессия выравнивает оценки между подразделениями. HR фасилитирует.

## Связь с компенсацией

| Рейтинг | Премия (годовая) | Грейд |
|---------|------------------|-------|
| A | 120–150% от целевой | Приоритет на повышение |
| B | 100–120% | Возможно повышение |
| C | 80–100% | Без изменений |
| D | 0–50% | ПИР на 90 дней |
| E | 0% | Решение о продолжении |

## Практический элемент: шаблон цели SMART

**Цель:** Снизить отклонение КСГ на участке «Северный» с 18 до 7 дней к 30.09.2025.  
**Метрика:** Дни отклонения от базового КСГ.  
**Вес:** 30%.  
**Шкала:** 5 — ≤ 7 дней; 4 — 8–12; 3 — 13–18; 2 — 19–25; 1 — > 25.
